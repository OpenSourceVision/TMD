# 📝 TMD — 极简 Markdown 预览与阅读器

TMD (Tiny Markdown Document) 是一款专为 Android 打造的原生 Markdown 阅览与管理工具。它采用极致的 **GitHub 风格美学设计**，提供纯粹、流畅的 Markdown 文档阅读与时间归档管理体验。

---

## ✨ 核心特性

- 📅 **按日期时间归档**:
  - **日期分组展示**: 首页书库按导入日期（如 `2026年07月22日`）自动分组归档展示。
  - **清晰导入时间**: 列表每份文档均展示具体的精确导入时间（`yyyy/MM/dd HH:mm`），方便快速检索回顾。
- 📥 **无缝导入与智能判重**:
  - **静默无缝导入**: 从本地导入 `.md` / `.txt` 文件后保留在当前首页，不中断浏览流，并伴有轻量级导入反馈。
  - **智能判重机制**: 导入已存在的文档时自动去重并更新记录，防止库内产生重复冗余文件。
- 🎨 **GitHub 风格主题**:
  - 精确还原 GitHub 浅色（Light）与深色（Dark）配色方案。
  - 支持 **系统跟随切换**、**手动固定深色** 或 **手动固定浅色** 三种主题模式。
- 📖 **原生高效渲染引擎**:
  - 基于 Jetpack Compose 纯原生组件的高效 Markdown 解析与排版。
  - 完美支持多级标题、粗体斜体、代码块、引用段落、超链接、列表以及内联代码高亮。
- 📂 **便捷文档管理**:
  - **滑动删除 (Swipe-to-Dismiss)**: 列表内向左或向右滑动即可快速移除文档。
  - **秒级实时搜索**: 实时对标题与文档内容进行检索。
  - **星标收藏**: 快速将重要文档加入收藏夹，独立归纳常用内容。

## 🛠️ 技术栈

- **构建系统**: Gradle (Kotlin DSL)
- **开发语言**: Kotlin 与现代协程 (Coroutines & Flow / StateFlow)
- **UI 框架**: Jetpack Compose (Material Design 3)
- **数据持久化**: Room Database (KSP)
- **架构模式**: MVVM (Model-View-ViewModel)
